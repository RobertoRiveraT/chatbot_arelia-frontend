Tutorial - Step 2: Registering

When a user registers, the app sends a POST request to the /register endpoint with the provided email and password. If the user doesn't already exist, the backend hashes the password (using bcrypt) and stores the new user in the database.

Upon a successful response, the app automatically logs the user in, saves the received JWT token in localStorage, and redirects them to /chatbot/chat, following the same behavior described in Step 1.
